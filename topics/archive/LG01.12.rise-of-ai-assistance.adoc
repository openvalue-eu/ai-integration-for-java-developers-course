=== Strengths vs Limitations
[cols="1,1"]
|===
|**Capabilities** |**Limitations / Risks**
|Rapid boilerplate generation |Hallucinated APIs or insecure patterns
|Contextual test suggestion |Data leakage (sending code snippets)
|Cross-language translation |Poor architectural reasoning
|Learning aid / doc summarization |License & IP uncertainty
|===

=== Categories of AI-Assisted Tools
* **Coding copilots** – GitHub Copilot, Codeium, Tabby
* **Documentation & comment generators** – Mintlify, Cody
* **Testing & QA tools** – CodiumAI, DeepCode, Testim
* **Refactoring assistants** – ChatGPT plug-ins, IntelliCode
* **DevOps/SRE automation** – AIOps dashboards, predictive alerts
