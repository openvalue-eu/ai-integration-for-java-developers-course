<dependency>
    <groupId>org.tribuo</groupId>

=== Tribuo's Type Safety

[.notes]
--
**Instructor Notes:** Emphasize this as key differentiator. Show compile-time safety benefits.
--

**Type Safety Features:**
[%step]
--
- Generic types: `Model<Label>`, `Prediction<Label>`
- Compile-time checking of feature types
- Type-safe evaluation metrics
- Prevents common ML errors at compile time
--

**Example Benefit:**
```java
// This won't compile - type mismatch
Model<Label> classifier = ...;
Model<Regressor> regressor = ...;
regressor = classifier; // Compile error!
```
