=== Installing GitHub Copilot Plugin
[.notes]
--
**Instructor Notes:** Walk through installation step-by-step. Show where to find it in Marketplace.
--

**Installation Steps:**
[%step]
--
1. Open IntelliJ → Settings → Plugins
2. Search "GitHub Copilot" in Marketplace
3. Install and restart IntelliJ
4. Sign in with GitHub account
5. Authorize Copilot access
--

=== Configuration Best Practices
[.notes]
--
**Instructor Notes:** Explain each setting's impact on suggestions and privacy.
--

**Key Settings to Configure:**
[%step]
--
- Enable/disable suggestions
- Suggestion delay timing
- Language preferences (Java, Kotlin)
- Telemetry and data collection options
- Inline vs. panel suggestions
--

**Common Concerns to Address:**
[%step]
--
- "Is my code sent to OpenAI?" - Yes, for suggestion generation
- "Can we disable telemetry?" - Yes, in settings
- "How to prevent sensitive code exposure?" - Use .copilotignore file
--

=== Understanding .copilotignore
[.notes]
--
**Instructor Notes:** Show example file. Emphasize importance for security-sensitive code.
--

**Key Points to Cover:**
[%step]
--
- Similar to .gitignore syntax
- Excludes files/folders from Copilot context
- Use for: config files, credentials, proprietary algorithms
- Example patterns: `*.properties`, `src/main/resources/application*.yml`
--
