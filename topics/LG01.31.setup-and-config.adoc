=== IntelliJ IDEA Settings

**Settings → Tools → GitHub Copilot**

**Key Options:**

* **Enable auto-completions**: Toggle inline suggestions
* **Show completions automatically**: How quickly suggestions appear
* **Suggestion display**: Inline vs. panel
* **Excluded files**: Ignore specific paths

=== Workspace-Level Configuration

**Create `.github/copilot-instructions.md`:**

```markdown
# Project-Specific Guidelines

## Code Style
- Use constructor injection for dependencies
- Prefer record types for DTOs (Java 21+)
- Use Spring Boot 3 patterns

## Naming Conventions
- Controllers: Use `*Controller` suffix
- Services: Use `*Service` suffix
- Repositories: Extend `JpaRepository`

## Frameworks
- Spring Boot 3.2+
- Java 21
- JUnit 5 with AssertJ
```

**Effect:** Copilot follows these guidelines project-wide

[.notes]
--
This is a powerful feature for team consistency
Ensures AI-generated code matches team standards
--

=== Custom Instructions for Chat

**Settings → Custom Instructions:**

```
When generating Java code:
1. Use Java 21 features (records, pattern matching)
2. Follow Spring Boot 3 conventions
3. Include error handling and logging
4. Generate JUnit 5 tests with AssertJ assertions
5. Use SLF4J for logging, never System.out

For REST endpoints:
- Use ResponseEntity
- Include OpenAPI annotations
- Handle validation with @Valid
```

[.notes]
--
Custom instructions persist across sessions
Improves consistency of generated code
--

=== Managing Copilot Suggestions

**Keyboard Shortcuts:**

* `Tab`: Accept entire suggestion
* `Ctrl+→` (or `Cmd+→`): Accept word-by-word
* `Alt+]`: Next suggestion
* `Alt+[`: Previous suggestion
* `Esc`: Dismiss suggestion

**Tip:** Browse multiple suggestions before accepting

=== Open Tabs: Context Control

**How It Works:**

* Copilot reads open editor tabs for context
* Related files = better suggestions
* Strategic tab management improves results

**Best Practice:**

```
Open tabs when working on feature:
- Controller (current work)
- Related Service
- DTO/Entity models
- Similar existing code
```

**Close unrelated tabs:** Reduces noise
