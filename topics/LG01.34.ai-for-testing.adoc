=== Test Patterns Copilot Understands

**1. Arrange-Act-Assert (AAA)**
```java
// Test user registration with valid data
@Test
void shouldCreateUserWithValidData() {
    // Copilot generates full AAA structure
}
```

**2. Given-When-Then (BDD)**
```java
// Given a user with valid credentials
// When they attempt to login
// Then they should receive an access token
```

**3. Parameterized Tests**
```java
// Test email validation with multiple invalid formats
@ParameterizedTest
// Copilot generates @ValueSource and test data
```
